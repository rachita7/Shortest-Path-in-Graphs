USER ACTIONS REQUIRED:
----------------------
1. Download the folder containing all the required files
2. Choose to use your own road.txt and hospital.txt. Or, choose to randomly create a hospital.txt
3. Run bfs1.java file for part ‘a’, ‘c’ and ‘d’, and run bfs_multi_final.java for part ‘b’
4. Enter value of ‘k’ when running bfs1.java
5. Output of the graph traversal is stored in RouteMap.txt
